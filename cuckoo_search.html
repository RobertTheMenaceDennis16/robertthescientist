Cuckoo Search Optimized Support Vector Machine

Research-in-Progress Portfolio Page

Project Overview

This research explores the application of Cuckoo Search optimization to enhance the performance of a Support Vector Machine (SVM) for fraud detection tasks. The goal is to improve classification metrics such as F1-score and Precision-Recall AUC by tuning hyperparameters using a metaheuristic approach.

Research Objective

To compare the performance of a Cuckoo Search–optimized SVM against baseline machine learning models including Random Forest, Gradient Boosting, and a standard SVM, using credit card fraud detection data.

  • Evaluate classification performance using F1-score
  • Measure Precision-Recall Area Under Curve (PR-AUC)
  • Assess model robustness using cross-validation

What is Cuckoo Search?

Cuckoo Search is a nature-inspired metaheuristic optimization algorithm based on the brood parasitism behavior of some cuckoo species. It uses Lévy flights to explore the search space and replace worse solutions with better ones iteratively.

  • Each solution represents a "nest"
  • New solutions are generated via Lévy flight random walks
  • Inferior solutions are replaced probabilistically
  • Best solutions are preserved over iterations

Support Vector Machine (SVM)

Support Vector Machines are supervised learning models used for classification tasks. They work by finding an optimal hyperplane that separates data points of different classes with maximum margin.

  • Effective in high-dimensional spaces
  • Uses kernel functions (linear, RBF, polynomial)
  • Sensitive to hyperparameters like C and gamma

Preliminary Results

  • Random Forest achieved strong baseline performance in F1-score.
  • Gradient Boosting showed lower PR-AUC compared to other models.
  • Baseline SVM underperformed without hyperparameter tuning.
  • Cuckoo Search–optimized SVM improved classification metrics significantly.

Evaluation Metrics

  • F1-score: Harmonic mean of precision and recall
  • PR-AUC: Area under the precision-recall curve
  • Precision: Correct positive predictions / total predicted positives
  • Recall: Correct positive predictions / actual positives
