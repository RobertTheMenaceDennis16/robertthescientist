Cuckoo Search Optimized Support Vector Machine

Research-in-Progress Portfolio Page

Project Overview

This research explores the application of Cuckoo Search optimization to enhance the performance of a Support Vector Machine (SVM) for fraud detection tasks. The goal is to improve classification metrics such as F1-score and Precision-Recall AUC by tuning hyperparameters using a metaheuristic approach.

Research Objective

To compare the performance of a Cuckoo Search–optimized SVM against baseline machine learning models including Random Forest, Gradient Boosting, and a standard SVM, using credit card fraud detection data.

Methodology

  • Data preprocessing and normalization
  • Baseline model training (SVM, Random Forest, Gradient Boosting)
  • Hyperparameter tuning using Cuckoo Search algorithm
  • Model evaluation using F1-score and PR-AUC
  • Cross-validation to ensure robustness

Technologies Used

Python Scikit-learn Pandas NumPy Matplotlib Machine Learning Optimization Algorithms

Preliminary Results

  • Random Forest achieved strong baseline performance in F1-score.
  • Gradient Boosting showed lower PR-AUC compared to other models.
  • Baseline SVM underperformed without hyperparameter tuning.
  • Cuckoo Search–optimized SVM improved classification metrics significantly.

Future Work

  • Experiment with additional metaheuristic optimization techniques
  • Test on larger and more diverse datasets
  • Deploy model as a web-based fraud detection tool
  • Integrate real-time prediction capabilities

About This Research

This project is part of an ongoing effort to apply machine learning and optimization techniques to real-world financial datasets. It demonstrates the use of evolutionary-inspired algorithms to improve predictive modeling performance.
